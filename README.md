## ✨ Features

- **Modern UI/UX** - Clean, professional design with smooth animations
- **Responsive Design** - Optimized for all devices (mobile, tablet, desktop)
- **Interactive 3D Elements** - React Three Fiber powered hero section
- **Smooth Animations** - Framer Motion for fluid transitions
- **Command Palette** - Quick navigation with `Ctrl/Cmd + K`
- **Project Showcase** - Interactive project cards with detailed modals
- **Skills Visualization** - Animated progress bars
- **Contact Form** - Easy way to get in touch
- **Social Integration** - Direct links to GitHub, LinkedIn, Instagram

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/roshan-1205/portfolio.git
cd portfolio
```

2. Install dependencies
```bash
npm install
```

3. Start development server
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
npm run build
```

The optimized build will be in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
